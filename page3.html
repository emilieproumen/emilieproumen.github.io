bouilloire éco-conçue

type

éco-conception d'un produit électro-ménager

année

2023

logiciels

SimaPro

Dans le cadre de ce projet, nous avons choisi un électro-ménager afin de l’éco-concevoir. Le produit choisi est une bouilloire électrique Silvercrest.
La première étape était la réalisation d’une analyse de cycle de vie. Pour ce faire, nous avons étudié les différents scénarios d’utilisation et décidé celui qui était le plus représentatif afin d’en déduire une unité fonctionnelle et un flux de référence.
Nous avons ensuite effectué des mesures de consommation électrique en cas classique d’usage. Puis nous avons démonté l’objet afin d’effectuer un inventaire de toutes les pièces, de les peser et de déterminer leurs compositions.
Une fois nos données en poche, nous avons modélisé la bouilloire sur le logiciel Simapro. Cette étape nous a permis d’identifier quelles étaient les sources d’impacts environnementaux majeures, et sur quels leviers nous allions travailler.
Nous avons donc proposé trois solutions (partielle, réaliste, ambitieuse) adaptées à ce constat. Nous avons modifié le nombre de pièces, les matériaux, mais surtout cherché une solution permettant d’éviter le plus gros impact négatif: la surconsommation électrique. Nous avons donc réfléchi à une proposition de bouilloire rétractable, adaptable à la quantité d’eau nécessaire à l’usage (une tasse pour le thé, un litre pour les pâtes), afin de pallier les surconsommations électriques.
